# 🎉 Bingo Bash Web 

```https://bingobash.xyz/info```

A sleek and interactive online portal designed for JY Tournaments' **Bingo Bash** Events!

Built with **React** and powered by **Google Sheets**, and **Firebase Tools**, this webpage ensures a smooth and engaging experience.  

## 🚀 Features  

### 🔹 Live Bingo Board  
- Displays a **Bingo grid** dynamically updated.  
- Winning tiles receive a **green tint** for better visibility.  


### 🔹 Admin Controls
- Ability to **manually mark** grid sections as won.  
- Updates reflect **instantly** across all public displays.  

### 🔹 Mappools Page 
- Displays **mappools** for each round.
- Clean UI for easy navigation.  

### 🔹 Info Page ℹ️  
- A dedicated page displaying **important event details**.  
- Sections for **rules, schedule, and player information**.  
- Auto-refresh support for **live schedule updates**.  

### 🔹 Firebase Hosting ☁️  
- Ensures **fast, scalable, and reliable** performance for all users.  

### 🔹 Mobile-Friendly 📱  
- Optimized layout for **both desktop and mobile devices**.  


## 🏆 Future Improvements

- **Sound effects** when a Bingo is achieved.
- Homepage improvements  
- Automatic win detection in the **Public Display**.
